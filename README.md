# Wordle Warriors – Score Calculator 

A simple and interactive **Wordle score calculator** that helps players calculate and track their performance in the game **Wordle Warriors**.

---

## Live Demo

 [Click here to try the app](https://wordlewarriors.netlify.app/)

---

## Features

* Calculate total Wordle score instantly
* User-friendly interface
* Fast and lightweight
* Accurate score calculation
* Responsive design for desktop and mobile

---

## Technologies Used

* HTML
* CSS
* JavaScript
  
---

## Future Improvements
